# How to use Colors

  Colors are defined in SASS and are leveraged by applying specific classes to elements.

  - Use the className `u-bg{color}{shade}` to style an element with a specific background color.
  - Use the className `u-color{color}{shade}` to style an element with a specific color.

For example, if I wanted to set the background of a div to Primary6 and the foreground to Neutral3, I would use the following:

```jsx
<div className="u-bgPrimary6 u-colorNeutral3">
  <p>This is a div with a primary background and neutral foreground.</p>
</div>
```
