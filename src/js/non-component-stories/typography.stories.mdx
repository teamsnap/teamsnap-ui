# Type Scale

  TeamSnap UI uses a [modular scale](https://www.modularscale.com/?14&px&1.125) with a ratio of 1.125.

  The scale is applied to headings and base type, but type sizing can also be  applied via utility classes if tweaks need to be made without impacting semantic hierarchy.

## Responsive Sizing

In cases where font size needs to scale based on viewport size, font sizes can be applied independently at each breakpoint.

<Canvas>
  <div style={{
    display:'grid',
    gridTemplateColumns: 'repeat(4, 1fr)',
    gridColumnGap: '16px'
  }}>
    {['xs', 'sm', 'md', 'lg'].map((breakpoint) => (
      <div>
        <h3><code>{breakpoint}</code> breakpoint </h3>
        <div className="u-spaceBetweenSm">
          {['Xs', 'Sm', 'Base', 'Md', 'Lg', '1x', '2x', '3x', '4x', '5x'].map((size) => (
            <div className="u-bgNeutral2 u-padMd">
              <p className={`u-${breakpoint}-fontSize${size}`}>u-{breakpoint}-fontSize{size}</p>
            </div>
          ))}
        </div>
      </div>
    ))}
  </div>
</Canvas>
