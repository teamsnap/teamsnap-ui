# Type Scale

  TeamSnap UI uses a modular scale with a ratio of 1.125. This small ratio is an ideal scale for utility-based apps that have dense content.

  Select levels of the scale apply to headings and base type, but a wider range of scale levels are available via utility classes if tweaks need to be made without impacting semantic hierarchy.

  Headings have responsive sizing applied so at small screens the larger sizes will automatically scale down.
